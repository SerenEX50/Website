# Bank Mock Website

## Overview
A mock banking website designed for educational and demonstration purposes. This project simulates the basic functions of an online banking system without handling real financial transactions.

The platform allows users to register accounts, log in securely, manage virtual balances, and view transaction records. It also includes a developing fake money transfer system with QR code payment simulation for practicing digital banking concepts.

This project helps developers and users understand how online banking systems work while providing a safe environment for learning web development, database integration, and authentication systems.

---

## Features
- User registration and login authentication
- Virtual account balance management
- Transaction history tracking
- Fake money transfer system *(In Progress)*
- QR code payment simulation
- Real-time database updates

---

## Technologies Used
- HTML
- CSS
- JavaScript

---

## Purpose
The purpose of this project is to simulate the core features of an online banking platform for educational and practice purposes. It serves as a safe environment for testing web development concepts without involving real financial transactions.
